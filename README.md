**This is a nice small C subset compiler implemented in Elixir.**

## Installation

1. git clone https://github.com/hiphoox/qncc.git
2. cd qncc
3. mix escript.build

## Using

- nqcc file_name.c
- nqcc --help

## Examples
- $ nqcc ./examples/return_2.c
